PROYECTO HTML-CSS

Introducción

El proyecto consiste en crear mi portfolio para los módulos de Administración de Sistemas Gestores de Bases de Datos (ASGBD) e Implantación de Aplicaciones Web (IAW).

Descripción del proceso

Tras la planificación y maquetación mental llega el momento de abrir WebStorm para empezar a crear el proyecto con todos sus directorios y archivos organizados, incluida la implementación de Git, sumando a ésto último el registro y la sincronización del proyecto en GitHub.
El código HTML y el código CSS se han desarrollado de manera simultánea durante este proyecto para poder crear una buena maquetación del proyecto en función de los resultados obtenidos con el código de estilos, siguiendo el siguiente orden:

  1. Barra de navegación
  2. Primera pantalla de portada y las páginas IAW y ASGBD
  3. Contenido de la portada
  4. Subpáginas de los módulos
Una vez realizado todo lo anterior llegó el momento de redactar este apartado y el del primer proyecto del modulo ASGBD.

Dificultades afrontadas

  1. Creación de la barra de navegación
    • Posicionamiento de submenús
    • Browsersync de WebStorm para definir rutas absolutas
  2. Posicionamiento con adaptación a pantallas
  3. Redacción del contenido de la portada

Posibles mejoras

Se me ocurren varias ideas que se podrían desarrollar utilizando JavaScript como la creación de un menú para pantallas de Smartphone o un carrusel de imágenes para dar interactividad al portfolio.
En proyectos más extensos es conveniente añadir un menú de contenidos al principio, algo muy habitual en páginas web.
